
## Flask snabbdom demo

### Setup

- Install python and nodejs
- `pip3 install virtualenv`
- `virtualenv venv`
- `source venv/bin/activate`
- `pip3 install -r pip_freeze.txt`
- `npm install`
- `sudo npm install -g browserify`

### Run Server

- `./manage.py runserver`
- Goto [http://localhost:5000/counter/](http://localhost:5000/counter/)
